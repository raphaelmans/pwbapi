### Installation
`pip3 install virtualenv`
`python3 -m venv env`
`source env/bin/activate` or `env\Scripts\activate`
`pip3 install -r requirements.txt`

### Start Env
`source env/bin/activate` or `env\Scripts\activate`


### Start App
`uvicorn main:app --reload`


### Server API Access
`http://127.0.0.1:8000/`
`http://127.0.0.1:8000/docs`

### Python Version
`3.11`
